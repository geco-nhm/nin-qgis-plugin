4 Eksportere og importere prosjekt

For å eksportere prosjektet må programtillegget QFieldSync lastes ned. Prosjektet kan deretter eksporteres fra QGIS på én enhet til QField på en annen enhet ved hjelp av kabel eller via QFieldCloud.

4.1 Eksporter prosjekt ved hjelp av kabel

4.1.1 Koble de to enhetene sammen med en kabel

Du vil nå få en melding på enheten med QField.

4.1.2 Klikk på Allow

4.1.3 Klikk på ikonet for Configure Current Project

4.1.4 Klikk på OK

4.1.5 Klikk på ikonet for Package for QField

4.1.6 Klikk på Create

4.1.7 Åpne filutforsker og naviger til C:\Users\username\QField\export. Høyreklikk deretter på prosjektmappen og klikk på Copy

4.1.8 Åpne filutforsker og naviger til This PC\Galaxy Tab Active4 Pro 5G\Internal storage\Android\data\ch.opengis.qfield\files\Imported Projects. Høyreklikk deretter inni mappen of klikk på Paste

Prosjektet er nå klar til å åpnes på enheten med QField.

4.2 Importer prosjekt ved hjelp av kabel

Man kan også importere prosjektet fra QField til QGIS. Først må du sørge for at enhetene er koblet sammen via kabelen.

4.2.1 Klikk på ikonet for Synchronize Project

4.2.2 Klikk på ...

4.2.3 Naviger til This PC\Galaxy Tab Active4 Pro 5G\Internal storage\Android\data\ch.opengis.qfield\files\Imported Projects.

A. Klikk på NiN_kartlegging og deretter på
B. Klikk på Select folder

Skjermbilde

4.2.4 Klikk på Synchronize

Prosjektet vil nå importeres i QGIS.

4.3 Eksporter prosjektet gjennom QFieldCloud

Dersom du ønsker å eksportere via QFieldCloud må du opprette en konto på QFieldCloud.

4.4 Opprett konto for QFieldCloud

4.4.1 Gå til innloggingssiden for å opprette konto for QFieldCloud

4.4.2 Klikk på Register for å registrere ny konto

4.4.3 Fyll ut feltene for å opprette brukernavn, passord, osv. Klikk til slutt på Register for å opprette konto

4.5 Installer QFieldSync

Åpne QGIS og sørg for at du har tilgang på internett. Dette er nødvendig for å laste ned programtillegget.

4.5.1 Gå til Plugins i menylinjen og klikk på Manage and Install Plugins...

Det vil dukke opp et vindu.

4.5.2 Installer programtillegget

A. Søk etter “QFieldSync” i søkefeltet
B. Klikk på QFieldSync
C. Klikk på Install Plugin

Ikonene for QFieldSync vil nå dukke opp i verktøymenyen.

4.6 Legg inn prosjekt i QFieldCloud

4.6.1 Klikk på ikonet for QFieldCloud Projects Overview

4.6.2 Logg inn med QFieldCloud

A. Fyll inn brukernavn og passord
B. Klikk på Sign In

4.6.3 Klikk på ikonet Start New Project

4.6.4 Opprett et midlertidig passord

A. Fyll inn midlertidig passord
B. Klikk på Ok

4.6.5 Velg innstillinger

A. Huk av for Convert currently open project to cloud project
B. Klikk på Next

4.6.6 Velg prosjektnavn og filplassering

A. Fyll inn prosjektnavn
B. Beskriv prosjektet kort
C. Angi ønsket filplassering

4.6.7 Klikk på Create

4.6.8 Klikk på OK

Prosjektet vil nå være eksportert til QFieldCloud og det kan nå åpnes på den nye enheten.

4.7 Importer prosjektet gjennom QFieldCloud

Før du kan importere prosjektet i QGIS må du sørge for at endringene som er gjort i QField, er lastet opp til QFieldCloud (se 4.6 Legg inn prosjekt i QFieldCloud). Import av prosjektet til QGIS er kun mulig med internett-tilgang.

4.7.1 Klikk på ikonet for QFieldCloud Projects Overview

4.7.2 Logg inn med QFieldCloud

A. Fyll inn brukernavn og passord
B. Klikk på Sign In

4.7.3 Klikk på prosjektet (my_cloud_project)

4.7.4 Klikk på Synchronize Selected Cloud Project

Det vil nå dukke opp et nytt vindu.

4.7.5 Klikk på Prefer Cloud

4.7.6 Klikk på Perform Actions

4.7.7 Klikk på OK

Prosjektet vil nå bli lastet ned fra skya og bli åpnet i QGIS.
